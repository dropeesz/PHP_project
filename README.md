## 📌 Sobre o Projeto

O **Sistema da Caixinha** foi desenvolvido para auxiliar no controle financeiro de grupos que utilizam o modelo de "caixinha", permitindo o gerenciamento de **pagamentos de cotas**, **empréstimos**, **juros** e o acompanhamento do **saldo total disponível**.

Este projeto foi criado com fins de **aprendizado e desenvolvimento profissional**, aplicando conceitos de desenvolvimento web e banco de dados.

> 🤖 Durante o desenvolvimento, ferramentas de Inteligência Artificial foram utilizadas como apoio para esclarecimento de dúvidas, sugestões de implementação e otimização do processo de aprendizagem.

## 🚀 Como Executar o Projeto

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/dropeesz/PHP_project.git
````

### 2️⃣ Mova o projeto para o XAMPP

Coloque a pasta do projeto em:

```text
C:\xampp\htdocs\
```

### 3️⃣ Inicie o XAMPP

Ative os serviços:

* Apache ✅
* MySQL ✅

### 4️⃣ Configure o Banco de Dados

1. Acesse:

```text
http://localhost/phpmyadmin
```

2. Crie um banco de dados.

3. Importe o arquivo:

```text
database/caixinha.sql
```

### 5️⃣ Configure a conexão

Edite o arquivo de conexão com o banco:

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "nome_do_banco";
```

### 6️⃣ Execute o projeto

Acesse:

```text
http://localhost/PHP_project
```
